# Python 学习笔记与练习

这个仓库记录了我从零开始学习 Python 的过程，代码均基于 [B站 Python 教程] 学习并实践。

## 📁 内容结构

### 基础入门
- `hello world.py` - 第一个程序，环境测试
- `day 1.py` - 个人信息收集器 (input + 类型转换 + f-string)
- `day1 review.py` - if判断与逻辑运算符复习
- `grade_checker.py` - 成绩等级判定器 (if-elif-else)
- `guess_number.py` - 猜数字游戏 (while循环 + break + random)

### 字符串与列表专项
- `day3 string_ops.py` - 字符串基础操作 (索引、切片、find、replace、编码)
- `day3 string_tasks.py` - 字符串综合任务 (isdigit/isalpha/isalnum、split/join、len)
- `day3学习整合之数字分类器.py` - 数字分类器 (列表 + for循环 + 奇偶判断)
- `day4之我要认真了（真的吗）.py` - 列表与字符串综合复习
- `day4之我这个fw把东西全忘了.py` - 字符串方法全面回顾
- `day4之晚上小练一下.py` - 晚间的巩固练习
- `day5之太几把麻烦了，只能写个半成品.py` - 阶段综合练习
- `day6之早上最后练一遍列表.py` - 列表知识点最终复习

### 🎯 完整小项目
- `book_manager.py` - **图书管理系统** (综合应用：字符串解析、列表嵌套、数据统计、库存管理、补货功能)
